# Node ES6+ Workflow
* Template workflow for creating ES6+ node applications.
* Includes src folder and pre-built configurations to handle HTML files and css.

## Install Locally:
1. Clone this repository
`git clone https://github.com/nchan9/es10-default-workflow.git`

2. Install NPM
`npm install`

3. Generate dist folder
`npm run dev`

4. Verify successful installation & files have compiled by running the webpack server. This will open a browser window displaying "hello world" in red.
`npm start`
